# 🎵 Music Streaming Application

A full-stack **Music Streaming Application** built using the **MERN Stack (MongoDB, Express, React, Node.js)**.  
This project allows users to stream music while admins can manage songs and users through a dedicated dashboard.

---

## 🚀 Features

### User Features
- Browse and play songs
- Music player with controls
- User authentication (Login / Signup)
- Playlist support

### Admin Features
- Admin dashboard
- Upload songs with cover image
- Delete and manage songs
- View registered users
- Role-based access (Admin / User)

---

## 🛠 Tech Stack

**Frontend**
- React
- Tailwind CSS
- Axios
- React Router

**Backend**
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer (file uploads)
